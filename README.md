## Installation

### Snap
[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-black.svg)](https://snapcraft.io/twitz)

### Build dependencies
- meson
- ninja
- libhandy
- WebKit2

### Build
```bash
meson build
cd build
ninja install
```
